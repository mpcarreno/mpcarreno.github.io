# María Carreño - Academic Portfolio

Personal website showcasing some of my personal and professional portfolio

## Local Development

### Prerequisites

- Ruby and Bundler
- Node.js
- Git

### Setup

```bash
# Clone the repository
git clone https://github.com/mpcarreno/mpcarreno.github.io.git
cd mpcarreno.github.io

# Install dependencies
bundle install

# Run the server
bundle exec jekyll serve -l -H localhost --force_polling
```

Visit `http://localhost:4000` to see your site.

### Using Docker

```bash
chmod -R 777 .
docker compose up
```

Access the site at `http://localhost:4000`.
